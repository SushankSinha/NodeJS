## 1. Import Modules

### require

```javascript
const library = require('./library');
```

### exports

- By assigning it to object module.exports

```javascript
const car = {
brand: 'Ford',
model: 'Fiesta'
}
module.exports = car
```

- Second one is to add the exported object as a property of exports

```javascript
const car = {
brand: 'Ford',
model: 'Fiesta'
}
exports.car = car

//or

exports.car = {
brand: 'Ford',
model: 'Fiesta'
}

//for importing

const items = require('./items')
items.car
// or
const car = require('./items').car

//The first exposes the object it points to. The latter exposes the properties of the object it points to.
